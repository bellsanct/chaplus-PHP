# chaplus-PHP
対話エージェント構築サービスChaplusの対話APIをPHPで利用するものです。

```
$data = [
  "api_key" => "YOUR_API_KEY",
  "agent_id" => "YOUR_AGENT_KEY",
  "utterance" => "", //Please put message
];
```
には、利用登録を行った後対応した値を入力してください。
